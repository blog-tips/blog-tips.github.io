mtブログとwordpressブログの比較

CMSなんとかで有名な Movable Type と wordpress ですが、ブログの作り方としては、どちらがいいのでしょうか。私は mt ブログの方が作りやすいのですが、wordpress もさくさくなんでどちらもいい感じがします。

wordpressブログは再構築が必要ない

基本的に、Movable Type と wordpress ではどちらも同じような気もしますが、html のカスタマイズという点では、mt ブログの方が若干やりやすいという印象もあります。

mt は複数のブログを構築するのも簡単ですし、php 関連がさっぱりな私でも、直感だけでぶなんな感じに作成することができています。

wordpress でも、いくつかのブログを作っているのですが、若干カスタマイズの方が初心者簡単でないという印象もあったりします。php あれこれというより、独自タグあれこれがどうもわからなくって、、。

デフォルトでは動的ページになっているので、とりあえず、html での出力にしてみたりいろいろしたりしています。ただ、wordpress には再構築の必要がないという点は、やっぱ、ブログのボリュームが大きくなってくると魅力的かと思います。

ブログでは個別記事の説明タグが機械的に出力されてしまうのが嫌で、mt ブログをいろいろカスタマイズして個別に入力するようにしているんですが、同じことを wordpress でできないものか、現在手探り中です。

海外でも、国内でも、mt 派＜wordpress 派が多いように思いますが、私はどうも、mt の方が使いやすい派です。ただ、自動インストールに対応していない格安サーバーであれば、導入の簡単さから、どうしても wordpress をチョイスしてしまいます。
