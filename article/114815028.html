ＭＴブログの作り方

ＭＴブログは、「 Movable Type 」をレンタルサーバーにインストールして作成する、ＣＭＳと呼ばれているウェブサイトのことです。

最近、このＭＴブログの作り方に関する情報 - ＭＴブログ（Movable Type）の作り方 - を当サイトのサブドメインに追加してみました。

データベースの作成やインストールの方法など、「 Movable Type 」をレンタルサーバーに導入する際の簡単な手順を書いてます。

レンタルサーバーでのＭＴブログ作り方

ＭＴブログと似たようなＣＭＳには、WordPress や XOOPS などもありますが、どれもが人気のブログ構築ツールとなっています。

レンタルサーバーではホームページを作成することが多いと思いますが、ページボリュームの多いホームページを更新していくには、リンク関係の修正など何かと面倒くさい点が多いです。

そのような時、「 Movable Type 」のようなブログの構築ツールをサーバーにインストールすると更新作業も簡単にできるようになります。

ＭＴ構築用にサーバーを契約する際、MySQL などのデータベースを利用することができないと Movable Type を導入することができませんが、さくらインターネットのスタンダードや、チカッパレンタルサーバーなど格安サーバーでも対応しているところが多いです。

MTブログ可能な格安レンタルサーバー

格安レンタルサーバーのなかでも、バリュードメイン付属の XREA のサーバーが割と格安ですので、わたしはそこにインストールしてはＣＭＳブログ作りをしています。

通常の格安レンタルサーバーはデータベースが一個しか使えなかったり、マルチドメイン対応のものが少ないですけど、XREA はデータベースが MySQL ５個、PostgreSQL ５個に加え、マルチドメイン数も２０個まで利用することができるのでお得です。

さらに上位プランのコアサーバーもありますが、両方とも高機能なので使いやすいレンタルサーバーです。

ただ、格安・無制限ということは、大量ブログ作成のようなスパムチックなサイトも多くなってくる傾向にありますので、いろんなサーバーで比較・検討されてみてはいかがかと思います。

初心者でも導入が非常に簡単なのは、多少割高になっても、「Movable Type 自動インストール機能」がついているところを選択すると大変便利です。
