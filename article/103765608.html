無料ブログのメタタグ重複問題

無料ブログで何かと気になるのはメタタグの重複エラー問題なのではないかなと思います。わたしも最近はメタタグの重複をどうすればいいのか困っています。

シーサーブログでメタタグ重複の場合

無料ブログのなかでもシーサーブログのメタタグ重複の場合はわりとなんとかある程度は回避できそうな予感がしていますが、完璧にはやはり無理っぽいです。どうしても、過去ログやカテゴリページでのメタタグの重複が生じてしまいます。

ただ、シーサーブログの場合、タイトルタグをカテゴリ名にするとかってすれば何とかある程度は回避できる可能性が高いです。けれども、やはり、同じカテゴリページでは同じカテゴリ名がついてしまうので、多少は重複してしまいがちになります。

title タグの重複回避方法

title タグの回避方法としては、ブログの独自タグを利用して記事ごとにタイトルを違うふうにカスタマイズするといいと思います。シーサーブログをはじめほとんどのブログサービスではデフォルトで記事タイトルとブログタイルが表示されるので、違った感じにはなるはずです。

記事タイトルをブログタイトルに変更

description タグの重複回避方法

description タグの場合、わたしはブログ説明のまえに個別記事名が表示されるように設定しています。「個別記事名。ブログ説明」という感じで違うふうに設定しているんで、いちおうは description タグの重複をある程度は回避できています。

シーサーブログの場合のhtmlとしては、デフォルトの description タグの content=" のあとに <% if:extra_title %><% extra_title %>。<% /if %> を追加するだけです。

ちなみに mt ブログの場合は、もしタグ機能をつかっていないなら、そこに説明文を個別に設定するといい感じに仕上がると思います。

mt の個別記事 description のブログカスタマイズ

メタタグの重複の種類

メタタグの重複でもわたしがエラーになりやすいのはタイトルタグと説明部分のメタタグです。それ以外に何があるかといえば何もないんで、それしか重複のしようもありませんが、このふたつは結構やっかいです。

無料ブログによっては独自のタグがあるので、それを利用すればいいとは思いますが、シーサーブログの場合は過去ログでそれぞれ個別に日付を設定できればいいと思うのですが、その独自タグがちょっとみあたらないので探している最中です。

Seesaaブログ独自タグの基本文法

日付ごとに設定できれば、それぞれ個別のブログ説明や記事タイトルを設定できるので重複することはないと思います。

けれども、結局のところメタタグが重複したからといってそれほどは影響ないっぽい気がします。

ウェブマスターツールなんかでエラーが数百とかって出てるとなんかやばいかなって思いますが、そんなに影響している感じはなさそうです。
