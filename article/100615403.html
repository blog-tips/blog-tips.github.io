リンクタグのＨＴＭＬ

シーサーブログで使うタグのＨＴＭＬについてメモしておこうと思います。まず、リンクタグの <a href についてあれこれしようと思います。

この <a href の読み方ですが、わたしはずっとエーエッチレフと読んでいます。けれども、hrefというのはハイパーレファレンスの略のようなので、エーハイパーレファレンスとか、アンカーエッチレフとかいう人もいるようです。

シーサーブログの DTD の場合はaを大文字の A ではなく、小文字の a で記入するといいみたいです。

外窓表示のリンクタグやアンカーテキスト

<a href="http://"></a>

外窓表示にする場合はターゲットをいれるといいみたいです。ブログランキングのバナーなどは外窓にするといいのではないかなと思います。

<a href="http://" target="_blank"></a>

rel="nofollow"を入れるとこんな感じのタグになります。

<a href="http://" rel="nofollow"></a>

rel="nofollow"いりの外窓にするとこんな感じです。

<a href="http://" target="_blank" rel="nofollow"></a>

ＨＴＴＰのところにリンク先のＵＲＬをいれて、リンクしたいテキストをその間にいれるといいようです。ほかにもtitleなどいろいろといれることができると思います。

<a href="http://リンク先ＵＲＬ">リンクしたい文字</a>

このなかでリンクしたい文字はアンカーテキストとよばれているようで、検索エンジンはリンク先のＵＲＬのページをリンクしたい文字と認識するようなので、被リンクを得るときに大切になってくるようです。

文字のかわりに画像をいれるとこんな感じになりますが、このときの画像のalt属性にリンクしたい文字を入力しておくといいのではないかなと思います。

<a href="http://リンク先ＵＲＬ">画像</a>
