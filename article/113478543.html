CMSブログの作り方

　CMSというのは"コンテンツ・マネジメント・システム"の略です。

　記事や画像などコンテンツを管理するためのシステムのことを指しています。無料ブログサービスのログイン後の"管理画面"がそれにあたります。

　無料ブログとは違い、独自ドメインを取得してレンタルサーバーで作成する場合は、自分で何らかの管理ツールを用意しなくてはなりません。
　その際のツールとして"Movable Type"や"Wordpress"などのCMSを利用して作成している人が多いです。

個人利用でのCMSブログの作り方

CMSでブログを作る際、レンタルサーバー（有料）を借りて、独自ドメイン（有料）を取得し、CMSツール（無料）をサーバーにインストールしてからブログを構築することになります。

CMSブログの作り方といった場合、「サーバーへのCMSのインストール方法」と「テンプレートなどのカスタマイズ方法」に分けられます。

CMSにもいろいろありますが、Movable Type、もしくはWordPressで作るのが人気になっています。

そのほか、xoops というCMSも人気ですが、xoopsは本体をサーバーにインストールしたあと、さらに blog 機能のアプリケーションをインストールする必要があるので、MT や WordPress と比較すると導入のハードルが高いです。

  • Movable Type（MT,MTOS）
  • WordPress
  • Geeklog
  • XOOPS

自分でこれらのツールをダウンロードしてインストールすることもできますが、自動インストールに対応しているサーバーを選ぶと便利です。

CMSでウェブサイトを構築する際、商用の企業サイトの場合だと、ライセンスの購入でかなりな費用がかかる場合もありますが、個人で利用する場合はほとんどが無料で使用できます。

Movable TypeとWordPressのCMS比較

Movable Type

  • 再構築の必要あり
  • 詳細なカスタマイズ可能
  • 複数のブログ作成が可能
  • 手動でのインストールは難しい

WordPress

  • 再構築の必要なし
  • 動的なページ生成
  • インストールは簡単

人気のMovable TypeとWordPress でCMS比較をしてみると、どちらかというと、独自タグを利用できるMTで構築した方が高機能っぽい気もします。

けれども、Movable Type は再構築が必要になりますので、ブログのエントリーが増えてくるにつれて、再構築の時間が長くなってしまう傾向にあります。

その点、php で動的にページが出力される WordPress は再構築をする必要はないですが、サーバーへの負荷や検索エンジンへのインデックスなどの面では静的なページが生成される mt に利点があるような気もします。

一方、レンタルサーバーへインストールする際の難しさからいえば、WordPress の方が格段に簡単です。

レンタルサーバーが自動インストールに対応していれば、どちらでも大差はないですが、自動インストールに対応していない格安サーバーで構築する際は WordPress の方が簡単のはずです。

複数のブログを構築する際は、WordPress はブログの数だけインストールしなくてはいけないという欠点があるので、Movable Type が使いやすいです。

CMSブログの作り方の手順

自動インストールに対応していないサーバーでも、MYSQLなどのデータベースが利用できれば、Movable Type などを導入することができます。

MTブログ（Movable Type）の作り方

☆ 自動インストール非対応での作成手順

  1. 独自ドメインの取得
  2. サーバー契約
  3. サーバーにデータベース作成
  4. CMS のダウンロード
  5. CMS のインストール
  6. ログイン
  7. 記事投稿

ただ、Movable Type の場合、3～5の手順はけっこう難しい面があります。

一方、格安レンタルサーバーのなかでも、自動インストールに対応していて、容量もある程度の余裕のあるサーバーを契約すると、これらの手順をはぶくことができます。
いろいろ試しましたが、MTで複数のブログが作成できるマルチドメイン対応のエックスサーバーというのがおすすめです。

MTブログ可能な格安レンタルサーバー

☆ 自動インストール対応での作成手順

  1. 独自ドメインの取得
  2. サーバー契約
  3. 自動インストール
  4. ログイン
  5. 記事投稿

個人でも、Movable Type をビジネスや商用で利用する際にはライセンスの購入が必要になりますが、機能的には若干異なるものの、無料で利用できるオープンソース・プロジェクト版のレンタルサーバーだと無料で使えます。

どちらで作成するにしても、インストールした後のブログカスタマイズが一番難しいです。
