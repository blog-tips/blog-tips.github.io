グーグルビデオバーの作り方

このブログにも最近設置してみたグーグルのビデオバーですが、このビデオバーの作り方は何かと難しいものがあります。

というのも、英語なのでこんな感じかな？こんな意味なんだろう的に判断して設置するっていうことになってしまうのだろうと思います。

わたしの場合はこのビデオバーはほかのブログで設置していたものの、無料ブログの作り方のようなブログでは再生する動画がまったくなかったもので設置していませんでした。

けれども、最近は自分で無料ブログの作り方に関するビデオ動画を作成しだしたので、それをちょっと表示してみようと設置してみたしだいです。

考えてみれば、芸能ブログよりも無料ブログの作り方のようなものの方が再生する動画はたくさんできると思います。

たとえば、リンクタグの作り方からはじまって、ブログのカスタマイズ方法、記事の書き方とか、お気に入りボタンの作り方とか無数に動画を作成できるのではないかなって思います。

まだ、４つぐらいですが、これから動画の数をちょくちょく増やしていこうって思ってます。

Video Search Wizard でビデオバー作成

ところで、無料ブログにこのビデオバーの作り方ですが、わたしの方法をちょっと記事にしておきます。

たぶん、コンピューターに詳しい人からみればあんまりうまくないかもしれないので、自分でいろいろと試してみるといいと思います。

このグーグルのビデオバーの作り方はまず、タグをVideo Search Wizardでもらってきます。

以前まではグーグルアカウントを取得していないともらえなかったのですが、最近はどうかちょっとわかりません。ない人はグーグルでアカウント登録します。ヤフーのヤフーＩＤみたいなものですが、たいていは取得ずみだと思います。

最初に①のSearch Expressionというところに、動画のキーワードを入力したらいろいろと動画のリストが右側に表示されます。

半角コンマでくぎっていくつか設定することができます。

たとえば、あなたがアイドル芸能情報みたいな無料ブログをやっているとしたら、「長澤まさみ」みたいな感じです。

でもやっぱ新垣結衣もいいなって思ったら、半角コンマをつけて「長澤まさみ,新垣結衣」みたいな感じです。

プレビューボタンを押すと動画のリストが表示されるので、それでいいと思ったら②の欄に自分のブログのＵＲＬを入力します。

③のジェネレートボタンを押すとコードが出てくるので全部コピーしてもらってきます。

このコードを自分のブログに貼り付けるといいのですが、そのまま貼り付けてもたぶん、あんまりよろしくないと思います。

そこで、コードのコメント扱いになっている箇所をよみながら修正していくわけですが、だいたい、ビデオ動画のスタイルシートとビデオバー本体の部分、ビデオ検索機能の部分ぐらいにわけることができると思います。

ここで、スタイルシートなどはヘッドタグの部分に記入するのがやっぱいいと思うので、わたしはその部分を消去してこんな感じでヘッドタグのなかにいれちゃってます。

<link rel="stylesheet" href="http://www.google.com/uds/
solutions/videosearch/gsvideosearch.css" type="text/css" />

<link rel="stylesheet" href="http://www.google.com/
uds/css/gsearch.css" type="text/css" />

さらに、サイドバーに入れる際なんかはちょっと大きさとか微妙な感じになるので、シーサーブログのスタイルシートに

#videoBar-player .playerInnerBox_gsvb .player_gsvb {
width : 320px;
height : 260px;
}

を追加して大きさをサイドバーに収まるようにしています。この３２０とかって箇所を自分のブログの幅におさまるようにするといいです。

適当なところで、ＨＴＭＬのチェックをしながら修正していけば、ある程度バリッド対応できる形にはなるのではないかなと思います。

と、こんな感じでけっこうビデオバーの設置は難しいようです。

コード内のコメントを読みながら設置するとなんとなく意味がわかるのですが、いかんせん、英語なので、とりあえずあまり大事じゃない無料ブログでお試し感覚で設置しながら作っていくといいと思います。

ちなみに、ビデオバーも発リンクにカウントされるかもしれないので、あまり関連性のないビデオを表示させたり、たくさん表示させたりすると検索順位が下がると思います。

これはシーサーブログでしか試してないのですが、グーグルブログの場合はこんな手間をかけなくてもボタンひとつで簡単に設置できるのでおすすめです。
