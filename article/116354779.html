ブログ引越しの重複インデックスとSEO

以前の記事で、ブログを独自ドメイン化して、新規 url に引越ししたシーサーブログのSEO効果について書いていたのですが、その後、インデックス数、複合キーワードでの検索ヒット数、総アクセス数の方がのきなみ上昇中でいい調子なんです。

ブログ引越しの際のリダイレクト設定は特に何もせず、現在は古い url と新規独自ドメインの url が検索結果で混在しているままなのですが、特にペナルティーといった様子ではなさそうです。

独自ドメイン化でブログ引越しのSEO効果

以前のシーサーブログを独自ドメイン化して新規 url へとお引越しした詳細はこんな感じです。

ブログ引越しのSEO対策

あまりにアクセス数が少ないもので独自ドメイン化してみたのですが、半年ぐらい様子をみてみようと気長に思っていたものの、三日ぐらいでアクセスが数倍に上昇したので、びっくりしてしまった次第です。

ただ、そのアクセスの詳細を調べてみると、増加分のほとんどは独自ドメイン化によるSEO的な要因ではなく、ヤフー掲示板からたまたま張られた被リンクによる増加分の割合が大きかったようです。

とはいえ、検索エンジン経由の複合キーワードの幅も広がっているので、概ね、ブログを独自ドメイン化することによるSEO的なアクセスアップ効果はあるといってもよいかと思います。

気になっていたのは、古い url と新規の独自ドメイン url でのインデックスが重複してしまうので、何らかのペナルティーがあるものと思っていたのですが、それも今のところは特に何の影響もなさそうです。

リダイレクト設定も何もせず、ただ、シンプルな形でリンク先を新ブログの url へと張替えただけなのですが、序所にインデックスの方も新 url へと移行しつつあるようです。

引越しによるインデックス数の推移はこんな感じです。

独自ドメイン化前
旧 url：60
新 url：0

独自ドメイン化後
旧 url：90
新 url：69

旧 url のインデックス数が、独自ドメイン化の前後で 60 から 90 に上昇しているのは多少不可解なのですが、原因はよくわかりません。

現在のところ、トップページのインデックスは旧 url ではすでに削除されており、独自ドメイン化した新規の url でしかインデックスされていない状態となっています。

今後の理想的な展開としては、旧 url でのインデックスがトップページ以外も徐々に削除されていき、最終的にはすべて新規 url へと移行してくれればと思っています。

今後のインデックス推移の予測
旧 url：0
新 url：90

ぶっちゃけ、グーグルやヤフーの検索エンジンは賢いと思うので、シンプルな形で普通にブログの引越しをすれば「これ、引っ越したんだな」って察してくれると思うんです。

ただ、その際にも、サイトエクスプローラーやウェブマスターツールを利用すると、スムーズに行くのではないかなという気がします。

独自ドメイン化費用の方もすでに回収済みなので、これだったらもっと早くやっておけばよかったなと思うんですが、次は多少ページランクがついてるブログでも試してみようと思います。
