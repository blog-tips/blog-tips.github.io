ブログにヤフーカスタムサーチを設置

ブログを作成するとデフォルトで検索窓がついていますが、シーサーブログの場合はもともとヤフー検索と連動している検索窓になっているようです。

そのほかにも、自分でカスタマイズな検索窓を設置しようと思ったら、google や msn の検索窓、ヤフーのカスタムサーチを使用して作ってみるとオリジナルなサーチボックスが出来上がるようです。

今回はヤフーカスタムサーチをブログに設置してみました。

ブログにヤフーカスタムサーチの作り方

ヤフーカスタムサーチの作り方といっても、html タグを取得して貼り付ければいいだけなんですが、ヤフーIDを持っていたらすぐに作成できると思います。

この検索窓は「ヤフーサイトエクスプローラー」というウェブサイト運営者向けツールと連動しているので、html タグ取得の際はヤフーIDでそちらの方へログインします。

カスタムサーチ - Yahoo!検索

作り方が詳しく書いてあるので、そのとおりに記入するとタグをもらえます。

  • url などブログ情報を入力
  • 検索結果画面の設定
  • 検索窓の見た目のデザイン設定

ブログサービスによって文字コードはだいたい決まってますが、シーサーは Shift_JIS、ヤフーブログやＦＣ２は euc-jp になってるみたいです。

html ソースに貼り付ける必要があるので、検索窓を設置できるブログサービスは割と少なめではないかなと思います。

多少難しいのはロゴ画像ですが、自分のブログタイトルあたりをキャプチャー画像で切り抜いて jpg で保存すると簡単です。

キャプチャー画像の作り方

html タグを取得して完了ボタンを押すと、サイトエクスプローラーのカスタムサーチの設定一覧に作成したブログの検索窓が追加されるようです。

気になるのはスタイルシートが独立していないことですが、ブログのスタイルシートに追加してもいいかと思います。

<style type="text/css">～</style>で囲まれた部分が検索窓のスタイルシートなので、この部分を切り抜いて、ブログのCSSとまとめてもいいかと思います。

html のソース的にも、html4.01 と xhtml では終了タグを閉じたり、閉じなかったりのvalidあれこれがあるので、シーサーブログの場合は input の終了タグを />で閉じる方がいいようです。

ヤフーカスタムサーチはサイトエクスプローラーと連動しているので、サイト内検索でのインデックス数が少ない場合は、サイトマップを送信すると改善されるかもしれません。
